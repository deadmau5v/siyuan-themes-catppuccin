## About

[Catppuccin](https://github.com/catppuccin/catppuccin) is a soothing pastel theme with 4 flavors that aims to be the middle ground between low and high contrast themes.

This theme provides Catppuccin Latte and Mocha flavors for SiYuan Note, a privacy-first, self-hosted knowledge management software.

## Usage

### Manual Installation

1. Download the latest release from the [releases page](https://github.com/deadmau5v/siyuan-catppuccin-theme/releases).
2. Extract the downloaded archive.
3. In SiYuan, go to `Settings` >> `Appearance` >> `Theme`.
4. Click `Install Theme` and select the extracted theme folder.

### From GitHub

1. Clone or download this repository.
2. Copy the theme files to your SiYuan themes directory:
   - **Windows**: `%USERPROFILE%\SiYuan\conf\appearance\themes\`
   - **macOS**: `~/Library/Application Support/SiYuan/conf/appearance/themes/`
   - **Linux**: `~/.local/share/SiYuan/conf/appearance/themes/`
3. In SiYuan, go to `Settings` >> `Appearance` >> `Theme`.
4. Select `Catppuccin` from the theme list.

### Theme Switching

To switch between Catppuccin flavors:

- Set the base theme to `Light` for **Latte**
- Set the base theme to `Dark` for **Mocha**

## Customization

The theme follows SiYuan's CSS variables system, making it easy to customize colors and appearance. You can create custom variations by modifying the CSS variables in the theme file.

## Problems?
